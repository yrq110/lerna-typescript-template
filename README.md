# lerna-typescript-template

## Commands

```bash
# install project root deps
$ yarn install

# install packages deps
$ yarn run bootstrap

# running dev server for package-site
$ yarn run dev:site

# build commonjs and esm files with .d.ts
$ yarn run build

# lint package files
$ yarn run lint
$ yarn run lint:fix

# run jest test suites
$ yarn run test

# clean all deps
$ yarn run clean
```
